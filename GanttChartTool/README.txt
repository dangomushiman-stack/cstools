日付・数値軸切り替え + アイコン導入版 v2

使い方:
1. このzipの中身を、.csproj があるフォルダへすべてコピーしてください。
2. 既存ファイルは上書きしてください。
3. dotnet run を実行してください。

重要:
- MainWindow.xaml を必ず上書きしてください。
- 画面上部の「🔢 数値軸」ボタンで、日付軸/数値軸を切り替えます。
- 数値軸ON: 横軸は 0, 1, 2... になります。
- 数値軸OFF: 従来通り日付軸になります。
- AppIcon.ico と Directory.Build.props により dotnet run / build でもアイコンが組み込まれます。
